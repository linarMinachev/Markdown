# Инструкция по работе с Markdown

## Выделение текста

Чтобы выделить текст курсиово, необходимо обрамить его звездочками (*).
Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**).
Например, **вот так**
